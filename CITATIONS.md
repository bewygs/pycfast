How to cite PyCFAST
===================

If you use PyCFAST in your research, please consider citing it as follows:

```bibtex
@software{pycfast,
  author = {Benoit Wygas},
  title = {PyCFAST},
  year = {2025},
  publisher = {Zenodo},
  version = {0.1.0},
  doi = {},
  url = {},
  note = {Software release}
}
```
