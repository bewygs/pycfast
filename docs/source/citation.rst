How to cite PyCFAST
===================

If you use PyCFAST in your research, please consider citing it as follows:

.. code-block:: bibtex

   @software{wygas_2026_pycfast,
       author    = {Wygas, Benoît},
       title     = {PyCFAST},
       year      = {2026},
       publisher = {Zenodo},
       doi       = {10.5281/zenodo.18703351},
       url       = {https://doi.org/10.5281/zenodo.18703351}
   }
